Nitsche Says

A Unity game modeled with Maya using C#

This multi-player game represents the distractions of any ordinary classroom. The teacher issues a set of "commands" that the student must replicate; these commands are a series of lights that appear on the white board. The teacher uses AWSD keys to give commands, and the student responds using the keyboard arrows. The game has 4 rounds. Round one has no distractions, Round 2 introduces distractions that can be repelled by clicking them, round three causes student players to involuntarily doze off, and Round 4 includes a dream sequence.
