Health Matters

A Motion Design Project

This interactive piece seeks to display the nutritional value of the American meal in a meaningful, unique format. The project addresses the difficulties of redesigning information on a topic (health) that is scrutinized constantly. It demonstrates the art of drawing the eye to certain words and places in a video, and many other visual techniques that emphasize healthiness.

Nitsche Says

A Unity game modeled with Maya using C#

This multi-player game represents the distractions of the classroom when a student is trying to pay attention to the commands/directions given by the professor. The "student" is the is trying to fight off common distractions such as Facebook, texting notifications, and other class notifications on T-Square in order to follow the professor's commands. The commands are given in a 'Simon Says' format.

Duckie Dash

A Gameboy Game written in C

This game was created for Gameboy and features animated sprites, a tiled background map (1024 x 1024), a collision map, sounds, four levels, and a few other special components. Music and sounds were created using Audacity and WAV2GB, Sprites and Background were drawn from scratch using Usenti. This game was made for Gameboy using C.
